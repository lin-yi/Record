## 引用
　　使用[CocoaPods](http://cocoapods.org)可以很方便地引入AYCategory。Podfile添加AYCategory的依赖。

```ruby
pod "AYCategory"
```

## 简介
　　AYCategory中包含了大量UIKit和Foundation的便利category，可以极大地提升开发效率。
